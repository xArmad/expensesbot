Use this link to invite the bot with all recommended permissions:

```
https://discord.com/oauth2/authorize?client_id=1454287057747968111&permissions=137439340608&scope=bot%20applications.commands
```

**Permissions included (137439340608):**

## How to Use

1. Click or copy the invite link above
2. Select your server (FameStar | F4F)
3. Click "Authorize"
4. The bot will be added with all necessary permissions

## Alternative: Generate via Discord Developer Portal

If you need to regenerate or modify permissions:

1. Go to: https://discord.com/developers/applications/1454287057747968111/oauth2/url-generator
2. Under **SCOPES**, check:
   - ✅ `bot`
   - ✅ `applications.commands`
3. Under **BOT PERMISSIONS**, select the permissions you need
4. Copy the generated URL
